# 意思決定の記録

## 原則

- トップレベルの文書には**結論だけ**を書く（経緯・議論の詳細は書かない）
- 意思決定には**なぜその選択をしたのか（理由）を必ず併記**する
- 経緯・議論の詳細は検討ログ（別ファイル）に書き、トップレベル文書から参照リンクを張る

## 記録形式

トップレベル文書：

```markdown
- **<決定内容>**（<日付> 確定）
  - 理由：<なぜこの選択をしたか。比較した代替案があれば一言>
```

検討ログ（`<文書名>-検討ログ.md`）：

```markdown
## <日付> <議題>

### 議論の経緯

- <提案・反論・判断の流れを時系列で>
```
