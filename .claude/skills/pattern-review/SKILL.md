# レビュー役付き

## 手順

1. 作業 worker を起動し、成果物を `work/<案件名>/draft.md` に書かせる
2. 検査役 worker を別途起動し、成果物のパスと検査観点（正確性・網羅性・体裁など）を渡す
3. 指摘は `work/<案件名>/review.md` に書かせる
4. 重要な指摘は作業 worker に差し戻す（修正→再検査）。軽微なら上司が判断して確定する

## 注意

- 作業と検査は必ず別の worker にする（自己レビューは通さない）
- 差し戻しは最大2往復まで。収束しなければ上司が裁定する
